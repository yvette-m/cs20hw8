Ditto Public API


I choice the ditto Pokedex API.
The Pokemon API makes all Pokemon data avaiable in one location.
I specifically choice the ditto section of the API because he's cute.
(I really don't have another reason)
I found this API by looking up "fun APIs" from the website
https://medium.com/codex/15-fun-and-interesting-apis-to-use-for-your-next-coding-project-in-2022-86a4ff3a2742
The paramerater I choice was Ditto out of all the pokemon in the Pokedex
API other than that I sorted all of Ditto's information using javascript
and then pulled out and printed only relevant information such as
his name, id, abilities, and stats. I also pulled out his image to
display to the page.
Applications were this can be used: In a game, some type of Pokemon
style game. Or a website about pokemon - like some type of
sorting data website to see information about each pokemon.
